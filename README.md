# React Shop

An e-commerce application built with **React 19** and **TypeScript**.
It uses **Vite** as the build tool, **Tailwind CSS** for styling, **PocketBase** as the backend, and **Zustand** for state management.

## Main Technologies

- React 19
- TypeScript
- Vite
- Tailwind CSS
- PocketBase
- Zustand
- React Router DOM
